# Angular Basics: Intro to Angular Interceptors

## Project Summary
A foundational guide on using Angular Interceptors to manage HTTP requests and responses globally, covering scenarios like loading screens and header management.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Inspect network requests to see the interceptor log and modify outgoing traffic.
